Unidirectional Data Flow In React, application data flows unidirectionally via the state and props objects, as opposed to the two-way binding of libraries like Angular. This means that, in a multi component heirachy, a common parent component should manage the state and pass it down the chain via props. Your state should be updated using the setState method to ensure that a UI refresh will occur, if necessary.
